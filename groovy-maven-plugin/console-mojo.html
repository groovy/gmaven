groovy:console

Full name:

org.codehaus.gmaven:groovy-maven-plugin:2.0-SNAPSHOT:console

Description:

Open a Groovy console window.

Attributes:

  • Executes as an aggregator plugin.
  • Requires dependency resolution of artifacts in scope: test.
  • Since version: 2.0.

Optional Parameters

Name Type Since Description
classpathScope ClasspathScope 2.0 Include additional classpath from project artifacts in given scope. Scope can be one of: none, provided, compile, runtime, test
Default value is: none.
User property is: scope.
scriptpath List 2.0 Path to search for imported scripts. Supports -Dscriptpath=PATH,PATH property syntax.
User property is: scriptpath.
